# My-emoji-language
Hello everyone, this is my first language on python, lets relax with funny symbols!!
Now I describe commands, it's similar to Brainfuck
- 😂 — начало цикла
- 😭 — конец цикла
- 👍 — увеличить значение в текущей ячейке памяти на 1
- 👎 — уменьшить значение в текущей ячейке памяти на 1
- 👉 — сдвинуть указатель памяти вправо
- 👈 — сдвинуть указатель памяти влево
- 🤖 — вывести символ (ASCII) из текущей ячейки
- 🔢 — вывести число из текущей ячейки
- ⏬ — ввести число с клавиатуры в текущую ячейку
- ▶ — начало цикла: если текущая ячейка = 0, перепрыгнуть на команду после соответствующего 
- ◀ — конец цикла: если текущая ячейка ≠ 0, вернуться на команду после соответствующего
